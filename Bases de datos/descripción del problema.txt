Se pretende hacer una página web para las personas que les gusta leer y escribir libros.
Para ser usurio se necesita conocer:
nombre, apellido, correo electronico, descripción de si mismo, nombre de usuario, rol, foto y contraseña,  
cada libro que está en la página tiene un nombre, fecha de publicación, autor, descripción y categoria. 
a su vez la categoría puede ser:  acción, aventura, romance, fantasia, humor, poesia,
misterio, terror y religioso. un libro solo puede tener una categoria, pero en una categoria pueden haber muchos libros. un usuario lector puede leer muchos 
libros y un libro puede ser leido por muchos usuarios lectores, ademas un usuario escritor puede escribir muchos libros, pero un libro puede ser escrito
por un usuario escritor.
